# public/images

Binary image assets are intentionally excluded in this environment.

Use these logical references in code/content:
- `/images/DSC08797-Edit-scaled.jpg`
- `/images/DSC08898-Edit-scaled.jpg`
- `/images/DSC08963-scaled.jpg`
- `/images/MMP03629-scaled.jpg`
- `/images/Sans-titre-7.png`
- `/images/TROUVE-TON-COFOUN-3-1024x1024.png`

UI components render neutral placeholders instead of binary assets to keep builds text-only compatible.
